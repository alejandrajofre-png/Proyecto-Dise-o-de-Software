# Proyecto-Dise-o-de-Software

Objetivos del proyecto:
1. Mejorar el rendimiento académico en los usuarios mediante las herramientas de organización mediante la opción de registrar tareas.
2. Aumentar la capacidad de trabajo grupal en los usuarios durante el semestre mediante las herramientas colaborativas.
3. Aumentar la capacidad de organización individual en los usuarios durante el semestre mediante registros del estado de cada tarea.
